Pourquoi cette page ?

L'application stocke tes données à deux endroits dans ton navigateur : le localStorage (tâches, notes, liens) et l'IndexedDB (le handle du fichier pour la sauvegarde directe).

Ces données ne sont pas supprimées en vidant le cache classique ou en fermant le navigateur. Cette page te permet de les effacer proprement, par exemple pour repartir d'une installation vierge ou résoudre un problème de synchronisation entre le fichier et le navigateur.

Attention : ces actions sont irréversibles. Si tu n'as pas sauvegardé tes modifications dans les fichiers mesLiens.js et mesNotes.js, tes données seront perdues.
