Sauvegarde automatique

L'application sauvegarde automatiquement tes données dans le localStorage du navigateur à chaque modification.

Ces données persistent même en fermant le navigateur. Utilise l'export JSON pour créer une sauvegarde externe.

Attention : effacer le localStorage est irréversible. Exporte tes données avant si tu veux les conserver.

Export / Import

Exporter (JSON) Télécharge toutes tes données dans un fichier JSON.
Importer (JSON) Restaure tes données depuis un fichier JSON.
